Abstract

The Cybersecurity Quiz Application is an interactive, web-based educational tool developed using React and TypeScript. It aims to enhance cybersecurity awareness by providing a gamified learning experience. The application features multi-language support (English and Burmese), tiered difficulty levels (Beginner, Intermediate, Advanced), a time-based scoring system, and achievement awards (medals). Upon successful completion, users can generate and download a personalized certificate. This report provides a comprehensive overview of the project's objectives, architecture, design, implementation details, testing strategies, and potential future enhancements.

4. Features

  • Multi-Level Quiz: Offers beginner, intermediate, and advanced difficulty levels.
  • Multilingual Support: Available in English and Burmese.
  • Timed Questions: Each question has a 30-second timer with auto-submission.
  • Scoring and Medals: Users receive a score and are awarded medals based on performance.
  • Downloadable Certificate: A personalized certificate can be downloaded upon quiz completion.

5. Conclusion and Future Work

The Cybersecurity Quiz Application provides a solid foundation for an interactive e-learning platform. Future enhancements could include:

  • Expanding the question bank.
  • Adding more languages.
  • Implementing a user dashboard to track progress over time.
  • Integrating with a robust backend to persist user data and scores.
